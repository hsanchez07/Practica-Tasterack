# Practica-Tasterack
Sistema de gestión de datos de un restaurante desarrollado con MySQL.

## Descripción

Sistema de gestión de datos de un restaurante desarrollado con MySQL.

## Contexto

Proyecto realizado en clase en un grupo de 3 personas para trabajar la gestión de bases de datos mediante procedimientos almacenados, triggers, eventos y control de datos.

## Tecnologías utilizadas

- MySQL

## Cómo ejecutar

Ejecutar los archivos SQL en MySQL Workbench.
